### 📏 Rules
- 💪 주말, 공휴일 제외 1일 1솔 목표
- 🛑 문제 풀이 시간 1시간 초과 시 스톱
- 📝 자정까지 `PR` 올리기
- ✅ 주 최소 1회 JavaScript로 풀이
> 어려웠던 부분, 개선할 부분 PR 내용에 포함해서 같이 리뷰

### 🕶️ Conventions

**Issue** | `[플랫폼] 문제이름`  
e.g. [Programmers] 문자열 출력하기

**PR** | `YYMMDD 이름`  
e.g. 230712 수정

**Commit Message** | `내용 문제이름`  
e.g. Create 문자열 출력하기  
e.g. Refactor 문자열 출력하기

**File Name** | `YYMMDD_ProblemName`  
e.g. 230712_PrintString.js
